<br>

### 풀이

문자열 파싱과 자료구조를 이용하는 문제이다.

String으로 들어온 입력을 처음 "["와 "]"를 제거하고, StringTokenizer로 ","를 기준으로 나눠서 Deque에 넣어줬다.

Deque에 넣은 이유는 뒤집기를 할때 pointer를 이용하여 앞뒤를 표시하기 위함이다.

그리고 문제 내용 그대로 R,D연산을 해주면 끝.
